# aws-cdk-todolist-ui

AWS Amplify frontend for the appsync backend [aws-cdk-todolist](https://github.com/mmuller88/aws-cdk-todolist-ui)

# Deploy
1. Test And Deploy
```bash
yarn test
yarn deploy
```

1. Generate Exports && GraphQL Types
```
yarn generate-exports
yarn codegen
```

1. Run the frontend locally
```bash
yarn dev
```
